#  Washington D.C. Bike Rental Dashboard

Interactive Streamlit dashboard for analyzing Capital Bikeshare rental patterns (2011-2012).

##  Features

- **5+ Interactive Visualizations**: Hourly patterns, seasonal trends, weather impact, user analysis
- **5 Interactive Widgets**: Multi-select filters for year, season, day type, temperature, and hour range
- **Real-time Filtering**: All visualizations update dynamically
- **Statistical Analysis**: Confidence intervals, correlation matrices, comparative analysis
- **Data Export**: Download filtered datasets
